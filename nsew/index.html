Buey - NSEW Virtual Keyboard

NSEW Virtual Keyboard

Instructions

  • Use 8, 6, 2 and 4 (up, right, down, left) keys to change the active key. Press 5 (enter) to select it.
  • The selected character will be written to the form and the cursor will return to the central key (Mode).
  • Select the central key (Mode) to change the keyboard mode (lowercase, uppercase, symbols, numbers).
